C Programming FAQs (Korean)

[ English | Korean ]

Usenet comp.lang.c의 공식 FAQ (Frequently Asked Question)인 C Programming FAQs에 대한 한국어 판을 제공합니다.

원 문서에 대한 저작권은 Steve Summit씨에게 있으며, Steve씨의 허락을 받아 번역판을 제공합니다. 이 번역판에 대한 저작권은 문서의 첫부분에 있습니다.

이 문서는 on-line C FAQ 리스트와는 약간 다릅니다. 차이점와 이 문서의 특징은 다음과 같습니다:

  • 모든 답변들은 새 표준 C99에 해당하는 레퍼런스를 가질 것입니다. (문서의 Reference chapter에 있는 [HS2002], [C99], [ISO Rationale], 그리고 [SUS]를 보기 바랍니다.)
  • 오리지널 온라인 C FAQ 목록에 있는 모든 내용을 포함합니다.
  • 출판된 책, C Programming FAQs -- Frequently Asked Question (Addison Wesley, 0-201-84519-9)에 있는 몇몇 예제를 인용합니다.
  • 앞으로 새 C99 기능들과 다국어 지원을 위한 wide 문자셋, multibyte 문자에 대해서도 다룰 예정입니다.

이 곳에 있는 모든 문서들은 출력 품질은 만족할만한 수준이 아닙니다. 현재 내용을 추가하고 개선하는데 집중하고 있으며, 레이아웃이나 디자인 문제는 차후에 개선하도록 하겠습니다.

Files

1996년, 이 문서에 대한 작업을 처음 시작할 때 WYSIWYG이 아닌 문서 작업에 관심이 있었기 때문에, 이 문서는 LaTeX 형식으로 만들고 있습니다. 따라서 PostScript와 PDF 형태로 제공됩니다.

HTML 형태의 문서를 요구하는 분들이 많아서, 다시 이 문서를 HTML 형태로 만들었습니다. HTML 형식의 문서는 .ps 또는 .pdf 형식보다 좀 더 최신의 내용을 담고 있습니다. 검색도 가능하며, 문서 안에서 다른 곳으로 쉽게 이동하기 위한 하이퍼링크를 많이 달았기 때문에, HTML 형식의 문서를 먼저 보기를 권합니다.

  • 바로 가기 HTML (.html)
  • 다운로드 Postscript (.ps)
  • 다운로드 gzipped Postscript (.ps.gz)
  • 다운로드 bzipped Postscript (.ps.bz2)
  • 다운로드 PDF
  • 다운로드 gzipped PDF (.pdf.gz)
  • 다운로드 bzipped PDF (.pdf.bz2)

현재 이 문서는 teTex 버전 2.0.2와 HLaTeX 버전 0.991로 만들었으며, latex2html 버전 2002.2로 변환하고 있습니다.

Help Wanted

이 문서를 좀 더 좋은 품질로 만들기 위해 도움을 주고 싶다면, 아래 단락을 읽고 저에게 메일을 주시기 바랍니다.

LaTeX 소스로부터 좀 더 좋은 품질의 .ps 또는 .pdf 파일을 얻는 방법을 아신다면 제게 알려주시기 바랍니다. 안타깝게도 현재 .pdf 문서에서 한글(한국어)로 검색이 불가능합니다. 현재 dvipdfmx로 .pdf 문서를 만들고 있습니다.

이 문서에 있을 수 있는 버그나 오타가 발견되면 꼭 알려주시기 바랍니다. 무엇보다도 도움이 됩니다. ;-)
