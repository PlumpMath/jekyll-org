Scripts

제가 만든 모든 Emacs script들은 제 개인 GIT respository에서 받을 수 있습니다. 여기에서 직접 볼 수 있습니다. 또는 아래 명령을 통해 받을 수 있습니다:

$ git clone http://www.cinsk.org/git/emacs-scripts

제 설정 파일을 쓰려면, 먼저 $HOME/.emacs를 제거하고, emacs-script 디렉토리를 $HOME/.emacs.d로 변경하면 됩니다:

# 지우는 것보다 따로 백업해두는 것이 좋을 것 같습니다.
$ rm -rf $HOME/.emacs $HOME/.emacs.d
$ mv emacs-script $HOME/.emacs.d

Screencast (Demo)

vnc2swf와 x11vnc를 써서, Emacs를 쓰는 데모(screencast)를 만들었습니다. (새로 만든 데모부터는, provide online demonstration(screencast) on my emacs session. xvidcap과 and mencoder를 썼습니다.)

대부분 데모는 Macromedia Flash를 써서 만들었습니다. 안타깝게도 해상도 스케일은 변경할 수 없습니다.

  • A "Hello, world" Session
  • Comments Editing Features
  • Artist Mode
  • nXML Mode (editing HTML)
