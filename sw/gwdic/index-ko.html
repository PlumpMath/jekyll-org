About

gwDic은, 간단한 웹 사전입니다. 사용자가 원하는 대로 설정할 수 있어서, 여러분이 쓰고 있는 웹 사전을 쉽게 추가할 수 있습니다.

Features

  • Emacs와 비슷한 단축키 제공 -- 마우스 필요없음.
  • History -- 찾아본 단어들을 쉽게 관리, 저장
  • 스크립트 지원 -- 다양한 작업 지원 가능

Downloads

현재, 개발 버전만 CVS를 써서 제공합니다. 여기에서 바로 보시거나, 아래 명령을 써서 소스를 받을 수 있습니다: 

  $ cvs -d :pserver:anonymous@www.cinsk.org:/root login
  password: <RET>
  $ cvs -d :pserver:anonymous@www.cinsk.org:/root co -r rel-0_1 gwdic
  ...
  $ cvs -d :pserver:anonymous@www.cinsk.org:/root logout

Install

gwdic을 설치하려면 다음과 같은 패키지가 필요합니다:

  • GTK+ 2.x
  • libglade 2.x
  • dictd 선택사항
  • Mozilla Firefox -- gtkembedmoz

Makefile에서 GWDIC_PREFIX를 원하는 대로 수정한 다음 아래처럼 실행하면 됩니다: 

  $ make all
  ...
  $ su
  password: ****
  # make install
  ...
