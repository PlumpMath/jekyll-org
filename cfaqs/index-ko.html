C Programming FAQ (한글/한국어)

English | Korean

 

Usenet comp.lang.c의 공식 FAQ (Frequently Asked Question)인 C Programming FAQs에 대한 한글(한국어) 판을 제공합니다.

원 문서에 대한 저작권은 Steve Summit씨에게 있으며, Steve씨의 허락을 받아 번역판을 제공합니다. 이 번역판에 대한 저작권은 문서의 첫부분에 있습니다.

이 문서는 on-line C FAQ 리스트와는 상당히 다릅니다. 원 문서의 내용을 포함하며, 다음과 같은 차이가 있습니다:

  • 모든 답변들은 새 표준 C99에 해당하는 레퍼런스를 가질 것입니다. (문서의 Reference chapter에 있는 [HS2002], [C99], [ISO Rationale], 그리고 [SUS]를 보기 바랍니다.)
  • 오리지널 온라인 C FAQ 목록에 있는 모든 내용을 포함합니다.
  • 출판된 책, C Programming FAQs -- Frequently Asked Question (Addison Wesley, 0-201-84519-9)에 있는 몇몇 예제를 인용합니다.
  • 앞으로 새 C99 기능들과 다국어 지원을 위한 wide 문자셋, multibyte 문자에 대해서도 다룰 예정입니다.

Files

이 곳에 있는 모든 문서들은 출력 품질은 만족할만한 수준이 아닙니다. 현재 내용을 추가하고 개선하는데 집중하고 있으며, 레이아웃이나 디자인 문제는 차후에 개선하도록 하겠습니다.

  • 바로 가기 HTML (.html)
  • 다운로드 Postscript (.ps)
  • 다운로드 gzipped Postscript (.ps.gz)
  • 다운로드 bzipped Postscript (.ps.bz2)
  • 다운로드 PDF
  • 다운로드 gzipped PDF (.pdf.gz)
  • 다운로드 bzipped PDF (.pdf.bz2)

LaTeX 포맷으로 된, 최신 소스는 제 CVS repository에서 다운로드할 수 있습니다. 물론 이 내용은 검증되지 않은 내용이나, 아직 완성되지 않은 내용을 포함할 수도 있습니다. 

  $ cvs -d :pserver:anonymous@www.cinsk.org:/root login
  Logging in to :pserver:anonymous@www.cinsk.org:/root
  CVS password: <RET>
  $ cvs -d :pserver:anonymous@www.cinsk.org:/root co cfaqs
  ...
  $ _

또, 여기에서 직접 볼 수 있습니다.

Help Wanted

이 문서를 좀 더 좋은 품질로 만들기 위해 도움을 주고 싶다면, 아래 단락을 읽고 저에게 메일을 주시기 바랍니다.

현재 이 문서는 teTex 버전 2.0.2와 HLaTeX 버전 0.991로 만들었으며, latex2html 버전 2002.2로 변환하고 있습니다. 현재 dvipdfmx로 .pdf 문서를 만들고 있습니다. 안타깝게도 현재 .pdf 문서에서 한글(한국어)로 검색이 불가능합니다. 검색 가능한 PDF 문서를 만드는 방법을 아신다면 연락 바랍니다.

무엇보다도, 이 문서에 있을 수 있는 버그나 오타가 발견되면 꼭 알려주시기 바랍니다. 무엇보다도 도움이 됩니다. ;-)

Donation

문서 자체를 자세히 검토할 시간은 없지만 도움을 주고 싶으면, 문서 작업을 위해 기부하실 수 있습니다. 현재 갖고 있는 문서 및 책은 다음과 같습니다:

  • C Programming Language, 2번째 판, ISBN 0-13-110362-8
  • C: A Reference Manual, 5번째 판, ISBN 0-13-089592-X
  • C Programming FAQs, ISBN 0-201-84519-9
  • ISO/IEC 9899:1999 Programming Languages -- C, 2번째 판
  • ISO/IEC 14882:1998 Programming Languages -- C++, 1번째 판

다음 문서 및 책이 필요합니다:

  • 잃어버린 POSIX Programmer's Guide, ISBN 0937175730
  • 닳아진 Advanced Programming in the UNIX Environment, ISBN 0201563177
  • C Programming Language, 첫번째 판, 0-13-110163-3
  • ISO/IEC 9899:1990 Programming Languages -- C
  • Single UNIX Specification 버전 3, 또는 IEEE Std 1003.1 (POSIX), 또는 ISO/IEC 9945. (pdf 포맷 필요)
  • ISO/IEC 10646:2003, Information Technology -- Universal Multiple-Octet Coded Character Set (UCS)
  • The Unicode Standard, Version 4.0, Addison-Wesley, 2003. ISBN 0-321-18578-1
  • 이 외에도 도움이 될 만할 책 또는 문서는 언제라도 환영합니다.

꼭 위에서 언급한 버전이나 edition이 아니더라도 괜찮습니다. 특히 K&R이라고 알려진 "C Programming Language", 초판이 절실합니다.

또는, 여러분의 신용 카드로 일정 금액을 기부하실 수 있습니다. 이 페이지를 방문하기 바랍니다.
