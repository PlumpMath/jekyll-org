Scripts

Includes my start-up script, .emacs, all of my scripts are available from my CVS repository. You can browse it here, otherwise download it as in following method:

$ cvs -d :pserver:anonymous@www.cinsk.org:/root login
CVS password: <RET>
$ cvs -d :pserver:anonymous@www.cinsk.org:/root co emacs-scripts
$ cvs -d :pserver:anonymous@www.cinsk.org:/root logout
$ _

Note that .emacs itself is named as dot-emacs.el.
