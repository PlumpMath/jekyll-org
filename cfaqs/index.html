C Programming FAQs (Korean)

[ English | Korean ]

I provide Korean translation of C Programming FAQs, the official FAQ from usenet comp.lang.c which is written by Steve Summit.

Thanks for his permission, I was able to translate this document in Korean. If you need any copyright information about my translation, read a few front pages of the document.

This document is somewhat different than the original on-line C FAQ list. Here are some plans and it's features:

  • All answers will have appropriate references regarding to C99. (See the Reference Chapter for [HS2002], [C99], [ISO Rationale], and [SUS].)
  • All contents in the original on-line C FAQ list is already included.
  • Borrowed some examples in the book version, C Programming FAQs -- Frequently Asked Question (Addison Wesley, 0-201-84519-9) will be included.
  • Later, we will have many questions and answers regarding to new C99 features plus wide character set and multibyte characters

Note that any format type of this document is far from printing quality. Currently, I'm concentrating the contents itself, not design nor layout problem.

Files

Note that this document is far more than just translation of Steve's work. I've updated it from time to time to inject missing part (e.g. internationalization, new concepts in ISO/IEC 9899:1999, known as C99).

When I first started this work, I found myself being enjoyed with non-WYSIWYG document processing; especially LaTeX. Thus, the translation is provided as PostScript and Portable Document Format (PDF if you want).

Finally! By the public demands, I rebuilt HTML version of this document, The HTML version has more recent than .ps or .pdf version. And (of course) it's searchable. And I put more effort to make rich hyperlinks to each question. So, try HTML version first.
