About

gwDic is a simple web dictionary for everyday use. It is highly configurable, so that you can add your favorite web dictionary without difficulty.

Features

  • Emacs like short-cuts -- You don't need a mouse action to use gwdic.
  • History (words that you found) management
  • Script support -- virtually, you can do anything you'd like to do

Install

You need following packages prior to install gwdic:

  • GTK+ 2.x
  • libglade 2.x
  • dictd (Optional)
  • Mozilla Firefox -- gtkembedmoz

Modify GWDIC_PREFIX variable in Makefile if you want to install somewhere else than "/usr/local", then run make as following: 

  $ make all
  ...
  $ su
  password: ****
  # make install
  ...
